# E-commerce Website Test Plan

To verify that the e-commerce website functions correctly and meets business requirements.

## Scope
- User Registration
- User Login
- Product Search
- Shopping Cart
- Checkout Process

## Testing Types
- Functional Testing
- UI Testing
- Regression Testing
- Compatibility Testing

## Test Environment
- Browser: Chrome, Firefox
- OS: Windows 10/11

## Deliverables
- Test Cases
- Bug Reports
- Test Summary Report
