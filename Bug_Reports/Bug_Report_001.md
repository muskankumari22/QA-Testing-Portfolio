# Bug Report

## Bug ID
BUG-001

## Title
Login button remains disabled after entering valid credentials

## Environment
- Browser: Chrome
- OS: Windows 11

## Steps to Reproduce
1. Open the login page
2. Enter a valid username
3. Enter a valid password

## Expected Result
The Login button should become enabled and allow the user to sign in.

## Actual Result
The Login button remains disabled.

## Severity
High

## Priority
High

## Status
Open
